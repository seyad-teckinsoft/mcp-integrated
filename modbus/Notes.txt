###For  static IP ###
sudo nmcli connection modify "Wired connection 1" \
ipv4.method manual \
ipv4.addresses 192.168.250.4/24 \
ipv4.gateway 192.168.250.1 \
ipv4.dns "192.168.250.1 8.8.8.8"

sudo nmcli connection up "Wired connection 1"


### To rest IP address ### For Dynamic ip address#
sudo nmcli connection modify "Wired connection 1" ipv4.method auto
sudo nmcli connection modify "Wired connection 1" ipv4.addresses ""
sudo nmcli connection up "Wired connection 1"
sudo systemctl restart NetworkManager

Modbus data type 
1. Coil - Bitwise data (0/1) - Direction bi
2. Discrete Inputs - Bitwise datatype only used for (read by PLC).
3. Holding Register - 16 bit datatype (16)size -Direction bi.
4. Input Register - 16 bit data type only used for read by PLC.

Port -  502 Most widly used (1502 to avoid used privillage warnings).
